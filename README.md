# mjofetch 
> [!WARNING]
> This project was for my own porpuses and therefore has no customisation options besides changing the source code itself

![Demo Picture](https://github.com/mjo2804/mjofetch/blob/main/README-Demo.png)

mjofetch is a fastfetch alternative i wrote in rust with my fastfetch config.
## Dependencies
### Rust
fs - for checking the nix packages - it is only checking the binarys and not the dependencies tho

libc 
### Other
Pywal - For Color Scheme 

NixOS - For Package Display
## Install
Build it with Cargo and add it to path with 

```

cargo install --path .

```
